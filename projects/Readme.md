First Step: blink a LED
- Connect the wires, LED, and resistor as shown in the figure
- Connect Arduino Board to PC using USB cable
- In Arduino IDE, go to **Tool->Board**, select **Arduino AVR Boards -> Arduino Uno**.
- Click **Tools → Port**, Choose the port corresponding to your Arduino (Usually /dev/ttyACM0 or /dev/ttyUSB0)
- Copy the code from **mysketch.ino** to Arduino IDE
- Click the **Verify (✓)** button, If successful, you'll see: Done compiling.
- Click the **Upload (→)** button.
(note: please do make sure no short-circuit and always connect a resistor after LED)
